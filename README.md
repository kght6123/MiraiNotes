# Mirai Notes

どこでもブラウザさえあれば、Markdownで快適にノートが作成できることを目指しています。

ノートのテキストデータは、生のmdファイルでGoogle Driveや、GoogleTasksと同期します。

バックエンドはLaravel(PHP)、フロントエンドはVue.jsを採用しています。

PWAへの対応も予定しています。

---

## 主な機能

* Markdownメモ作成
* Googleドライブ同期
* オフライン編集
* オフライン検索
