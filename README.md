# OSS Notes

MarkdownのノートをWeb上で作成し、何処でも編集や参照ができることを目指しています。

ノートのテキストデータは、生のmdファイルでGoogle Driveや、GoogleTasksと同期します。

バックエンドはPHP（Bear.Sunday）、フロントエンドはVue.jsを採用しています。
